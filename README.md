# gohelpers

A collection of highly opinionated Golang helper packages.

## Contents

1. [`config`](./config/): A configuration package that reads a Golang struct tags using reflection and constructs flags.
2. [`logger`](./logger/): Thin wrapper on Slog logger that adds the logger to a context.
